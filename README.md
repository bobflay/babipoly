# Babipoly RFQ Generator

Professional Request for Quotation (RFQ) generator for the Babipoly board game.

## Overview

This script generates professional RFQ documents in multiple formats (HTML, Text, and PDF) from the structured data in `rfq.json`. The generated documents include your company logo and all product specifications formatted in a professional manner.

## Features

- Professional HTML document with embedded logo
- Clean plain text version
- PDF generation (optional)
- Detailed component specifications
- Quality requirements
- MOQ pricing tables

## Requirements

- Python 3.6 or higher
- `rfq.json` file (already present)
- `logo.jpg` file (already present)

### Optional Dependencies

For PDF generation:
```bash
pip install weasyprint
```

**Note:** If you don't install weasyprint, you can still generate HTML and text versions. To get a PDF, simply open the HTML file in a browser and print to PDF.

## Usage

### Basic Usage

Simply run the script from the command line:

```bash
python generate_rfq.py
```

This will generate three files:
- `rfq_babipoly.html` - Professional HTML document with logo
- `rfq_babipoly.txt` - Plain text version
- `rfq_babipoly.pdf` - PDF version (if weasyprint is installed)

### Opening the Generated Files

**HTML Version:**
```bash
open rfq_babipoly.html  # macOS
xdg-open rfq_babipoly.html  # Linux
start rfq_babipoly.html  # Windows
```

**Text Version:**
```bash
cat rfq_babipoly.txt
# or open in any text editor
```

**PDF Version:**
- If generated: Open with any PDF viewer

## Customization

### Updating Product Information

Edit `rfq.json` to modify any product specifications, components, or requirements. The script will automatically generate updated RFQ documents.

### Changing the Logo

Replace `logo.jpg` with your new logo file (keeping the same filename), or update the script to point to a different logo file.

## Output Structure

The generated RFQ includes:

1. **Header** with logo and RFQ number
2. **Product Information** section
3. **Component Specifications** with detailed tables:
   - Game Board
   - Playing Cards
   - Paper Money
   - Houses & Hotels
   - Player Tokens
   - Dice
   - Packaging
4. **Quotation Requirements** (MOQ levels and required information)
5. **Quality Requirements** checklist
6. **Legal Notice**

## Troubleshooting

**Issue:** PDF not generating
- **Solution:** Install weasyprint with `pip install weasyprint`, or print the HTML to PDF from your browser

**Issue:** Logo not appearing
- **Solution:** Ensure `logo.jpg` exists in the same directory as the script

**Issue:** Script fails to run
- **Solution:** Check that `rfq.json` exists and is valid JSON
